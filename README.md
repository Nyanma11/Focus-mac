# Focus-mac

Focusは、あなたの集中を助ける強力なデスクトップアプリです。特定のアプリやウェブサイトへのアクセスを遮断し、作業効率を向上させます。


## 特徴

| 特徴 | 内容 |
| :--- | :--- |
| **強力なブロック** | hostsファイルを書き換え、アプリ終了後も制限を継続することができます。 |
| **一時解除タイマー** | 設定時間のみ制限を解除。終了後は再ブロックします。 |
| **メニューバー常駐** | バックグラウンドで稼働し、作業を邪魔しません。 |


## インストール

### Releaseからダウンロード

[こちらのページ](https://github.com/Nyanma11/Focus-mac/releases)からダウンロード後、解凍してお使いください。初回起動時はgatekeeperにより起動がブロックされるため、システム環境設定の「プライバシーとセキュリティ」からFocusを承認してください。


## 使い方

1. **アプリの起動**

   `Focus.app`を起動します。起動した瞬間からブロックが有効になります。起動後はメニューバーに常駐します。

   >※ hostsファイルの書き換えを行うため、パスワードの入力が必要です。

2. **設定画面の呼び出し方**

   設定を変更したいときは、メニューバーにある「Focusのアイコン」をクリックしてください。プルダウンが表示されます。また、プルダウン内の「タイマーを表示する」をクリックすると、メインウィンドウが表示されます。

3. **ブラックリストの設定**

   メインウィンドウの「設定」タブで、制限したいアプリ名のシステム内部名（例：`Music`）や、制限したいサイトのURL（例：`youtube.com`）を入力して追加します。URLを削除することもできます。

   >* Hostsファイルの書き換えを行うため、パスワードの入力が必要です。

4. **制限の一時解除**

   メインウィンドウの「タイマー」タブで、任意のタイマー時間を入力して「一時解除スタート」をクリックします。タイマー作動中のみ制限が解除されます。

   >※ 解除開始時と終了時にはhostsファイルの書き換えを行うため、パスワードの入力が必要です。

5. **アプリの終了**

   メニューバー常駐アプリのためウィンドウを閉じてもアプリは終了しません。アプリを終了したいときはメニューバーのプルダウンメニューから終了してください。

   >※ 「アプリ終了後も制限を維持」がオフの場合、hostsファイルの書き換えを行うため、パスワードの入力が必要です。


## 注意事項

- 本アプリはシステムファイル（hosts）を編集します。
- 本アプリを削除した場合などで、制限が残ってしまった場合は下記の **緊急時の手動解除方法** を参照してhostsファイルをクリーンアップしてください。


## 緊急時の手動解除方法

アプリが起動できない、または削除してしまった等の理由でアクセス制限が残ってしまった場合は、以下の手順で制限の解除してください。
＞※ システムファイル（hosts）の編集を行います。

1. ターミナルを開き、以下のコマンドを実行します。

```Bash
# hostsファイルを編集モードで開く
sudo nano /etc/hosts
```

2. パスワードの入力が求められるので、半角で入力してください。

>※ 入力した文字は画面に表示されませんが、入力後、returnキーを押せば大丈夫です。

3. キーボードの矢印キーで移動し、youtube や twitter などが含まれる行を Control + K で削除します。

4. Control + O（保存）→ Enter → Control + X（終了）の順に押します。

5. 最後にキャッシュをクリアします。以下のコマンドを貼り付けて実行してください。

```Bash
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
```
6. パスワードの入力が求められるので、半角で入力してください。

>※ 入力した文字は画面に表示されませんが、入力後、returnキーを押せば大丈夫です。
